# Fotografía Brotea

Esta sección contiene recursos relacionados con el estilo fotográfico de Brotea, incluyendo guías y prompts para generar contenido visual que siga las directrices fotográficas de la marca.

## Contenido

- **[Prompts para Fotografía Detallados](./01_ESTILO_FOTOGRAFICO/photography_prompts.md)**: Colección de prompts detallados para generar fotografías profesionales que siguen el estilo visual de Brotea, con especificaciones sobre paleta de colores, iluminación, sujetos, ambientación y composición.

- **[Prompts para Fotografía Concisos](./02_CON_PERSONAS/concise_prompts.md)**: Versiones simplificadas y listas para usar de los prompts fotográficos, optimizados para herramientas de generación de imágenes.

## Estructura

### 01_ESTILO_FOTOGRAFICO
Recursos que definen el estilo fotográfico general de Brotea, incluyendo aspectos técnicos como iluminación, composición y tratamiento visual.

### 02_CON_PERSONAS
Recursos específicos para fotografías que incluyen personas, con directrices sobre poses, expresiones, diversidad y representación.

### 03_ABSTRACTO
Recursos para fotografías abstractas y conceptuales que complementan el estilo visual de Brotea.

## Características del Estilo Fotográfico

- **Paleta de Colores**: Tonos cálidos como dorados, terracotas y beiges suaves, complementados con toques de luz natural.
- **Iluminación**: Luz natural, preferentemente durante la "hora dorada" (mañana o tarde), con contraluces sutiles o luz lateral.
- **Sujetos**: Jóvenes (20-30 años) con actitud relajada, natural y auténtica, mostrando colaboración y trabajo en equipo.
- **Ambientación**: Espacios luminosos con toques informales como mesas de trabajo compartidas, cafeterías modernas o escritorios en casa, con accesorios tecnológicos cotidianos.
- **Composición**: Enfoque en gestos y expresiones, con primeros planos y planos medios que muestren emociones y detalles del entorno.

## Uso

Utiliza estos recursos para asegurar que todo el contenido fotográfico generado para Brotea mantenga una consistencia visual, reforzando la identidad de marca a través de un estilo fotográfico coherente y reconocible.
